Zero Query - Quick Start Guide

Installation

Download, and uncompress to any folder under your web server document root.

Enjoy it!

Customization

How to create new pages

Create a folder under 'content'. The name of the folder will be the name of your page. Under the page folder create a file named 'index.php'.

Thats all! The menu item will be created automatically and you can access your page at example.com/youpagename.

How to define Page Order

If you want to sort your pages, prefix the page folder name with a number + '.'. For example a folder '1. Home' will create a page at the first position of the menu, that can be accessed at example.com/home

How to exclude a page from main menu

Prefix the page name with '0.'

How to include page-level javascript / css

Yor page needs custom css or javascript? The only thing you need to do is create a css folder containing 'example.css' or a js folder containing 'example.js'

How to Build internal URL's

To include template images:

<?php echo TEMPLATE_PATH?>/images/xxxxxx.png Each page can render custom images that are not in the template. To include page images, prefix the image url with page path: Example:

<img src="<?php echo $this->path."/img/project2.png" ; ?>">
